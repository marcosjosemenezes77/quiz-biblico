Política de Privacidade

Última atualização: 30 de maio de 2026

1. Sobre este aplicativo

O Quiz Bíblico — Evangelho do Reino em Um Minuto é uma aplicação web gratuita, disponível em https://marcosjosemenezes77.github.io/quiz-biblico/, criada para que inscritos no canal do YouTube @evangelhodoreinoemumminuto possam testar seus conhecimentos bíblicos.

2. Quais dados são acessados

Para verificar se você já é inscrito no canal, o quiz solicita acesso à sua conta Google com o escopo:

  • youtube.readonly — leitura somente das suas inscrições no YouTube
Nenhum dado pessoal (nome, e-mail, histórico de vídeos, comentários ou qualquer outra informação) é coletado, armazenado ou transmitido para servidores próprios.

3. Como os dados são usados

O token de acesso obtido via Google OAuth 2.0 é usado exclusivamente para uma única chamada à YouTube Data API v3:

  • Verificar se a conta autenticada está inscrita no canal @evangelhodoreinoemumminuto

O resultado (inscrito ou não inscrito) determina apenas se o quiz é liberado para uso. O token não é salvo em banco de dados, servidor ou localStorage. Ele existe somente na memória da aba do navegador e é descartado ao fechar ou recarregar a página.

4. Compartilhamento de dados

Nenhum dado é compartilhado com terceiros. A aplicação é inteiramente estática — não possui backend, banco de dados ou servidor próprio. Toda a comunicação ocorre diretamente entre o seu navegador e os servidores do Google/YouTube.

5. Cookies e armazenamento local

O quiz não utiliza cookies. Nenhuma informação de identificação é gravada no localStorage ou sessionStorage do seu navegador.

6. Serviços de terceiros

Esta aplicação utiliza os seguintes serviços externos, sujeitos às suas próprias políticas de privacidade:

  • Google Identity Services — autenticação OAuth 2.0
  • YouTube Data API v3 — consulta de inscrições
  • Google Fonts — carregamento das fontes Bebas Neue e Montserrat
  • GitHub Pages — hospedagem da aplicação

7. Direitos do usuário

Como nenhum dado pessoal é armazenado por esta aplicação, não há dados a consultar, corrigir ou excluir. Você pode revogar o acesso concedido a qualquer momento em:

https://myaccount.google.com/permissions
